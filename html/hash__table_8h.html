Detailed Description

Simple hash table that maps integer keys to string values.

Author
Linus Hellstr√∂m, Noah Wassberg
Date
1 Sep 2022 Here typically goes a more extensive explanation of what the header defines. Doxygens tags are words preceeded by either a backslash \ or by an at symbol @.

We chose to use option_t as our fail management option, since it seemed convenient to save two values into and object that we simply can extract, and call Failure() if it fails.

The hash table does not take ownership of its keys and values.
